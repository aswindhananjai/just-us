## 🆘 Troubleshooting

**Can't add photos?**
- Make sure Cloudinary preset "memories" is created
- Check that signing mode is "Unsigned"

**Database errors?**
- Run the SQL schema in Supabase
- Check that the memories table exists

**App not loading?**
- Check the Vercel deployment at: https://vercel.com/dashboard
- Environment variables should be set
